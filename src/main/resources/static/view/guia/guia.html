2) Divisão arquivo de descrição

 O código do arquivo é divido em três partes, as anotações @Header, @Lexico e @Sintatico.

@Header

 Na anotação @Header é obrigatório informar três configurações: NomeProjeto, Pacote e Versao. A configuração Autor não é obrigatória, mas pode ser informada.

@Lexico

 Na anotação @Lexico é informado as regras léxicas para o analisador léxico. Regras que devem ser ignoradas precisam estar marcadas com a anotação @Ignorar. É necessário ter uma atenção no momento de posicionar as regras, pois a posição é importante e vai influenciar a classificação dos símbolos. Por exemplo, se for definido uma regra para Identificadores e logo abaixo for defindo uma regra para uma palavra reservada qualquer, essa palavra reservada não será classificada corretamente. Nesse caso será classificada como Identificador.

@Sintatico

 Na anotação @Sintatico é informado a gramática livre de contexto que será utilizada pelo analisador sintático LL1. Na gramática são aceitos Epsilon (£), Terminais (É o ID da regra léxica) e Não Terminais (Marcados com < e >).

3) Dúvidas sobre como implementar o arquivo de descrição?

 No momento do desenvolvimento do arquivo de descrição podem surgir algumas dúvidas, seja a respeito da sintaxe ou das expressões regulares. A ferramenta disponibiliza dois exemplos para você estudar, clicando em Exemplos é mostrado os exemplos: BNF (Arquivo descrição com a implementação da BNF) e Expressão Aritméticas Simples (Exemplo clássico utilizado em livros de compiladores). Os dois exemplos são LL1 e podem ser testados. Se houver dúvidas a respeito da implementação das expressões regulares, é possível utilizar o testador de expressão regular. Ele está visível na tela principal, no lado direito na Tab Testar RegExp.

4) Como vou baixar o projeto?

 Ao finalizar o desenvolvimento do arquivo de descrição é possível submeter ele a ferramenta. Para submeter o arquivo deve ser clicado em Executar, a ferramenta vai analisar o arquivo e apresentar na tela as informações sobre o analisador léxico e sintático.

6) O que acontece quando eu clicar em Baixar?

 Quando você quiser gerar o projeto é só clicar no botão Baixar, a ferramenta vai verificar novamente o arquivo de descrição e gerar os códigos na linguagem de programação Java, na versão 8, contendo o analisador léxico e sintático. Também é acrescentado ao projeto algumas classes utilitárias. O projeto gerado está configurado com todas as depêndencias necessárias para funcionar utilizando o Maven. Após baixar o projeto você pode importar ele para qualquer IDE que suporta Maven ou utilizar seu editor de texto preferido.
